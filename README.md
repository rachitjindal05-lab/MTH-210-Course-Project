# MTH-210-Course-Project
**Title: Parameter Estimation and Bootstrap Confidence Intervals for the Exponentiated Exponential Distribution.**

**Description:**

This project focuses on estimating the shape **($\alpha$)** and scale **($\lambda$)** parameters of an Exponentiated Exponential distribution based on a given sample dataset. Implemented in R, the core analysis utilizes the Newton-Raphson algorithm to computationally determine the Maximum Likelihood Estimators (MLEs) from the log-likelihood equations. To assess the reliability of these estimates, the project further constructs and compares 95% confidence intervals using both parametric and non-parametric bootstrap resampling techniques.
